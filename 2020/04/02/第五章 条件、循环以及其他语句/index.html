第五章 条件、循环以及其他语句

5.1 再谈print和import

print

5.2 赋值

序列解包（可迭代对象解包）

  • 将一个序列（或任何可迭代对象）解包，同时（并行）给多个变量赋值

链式赋值

  • 将多个变量关联到一个值

增强赋值

  • *=, /=, %=, +=, -= （就地修改）

    5.4 条件语句

布尔值

  • 在用作布尔表达式时，以下被视为假
  • 其他值都视为真（任何python值都被视为真值）

条件表达式

  • 可以视为C语言中三目运算符的python版本
  • 如果条件为真（if后面的语句），表达式结果为第一个值，否则返回第二个值

条件语句

  • 条件
    • 比较运算符
    • 布尔运算符

断言

  • assert充当检查点，使程序错误条件出现时立即崩溃，并打印AssertionError语句

    5.5 循环

迭代字典

并行迭代 zip

  • zip用于将“缝合”任意数量的序列
  • 返回值可用于迭代；若要查看其内容，可用list将其转换为列表

迭代时获取索引（枚举） enumerate
反向迭代 reversed / 排序后迭代 sorted

  • 二者可用于任何序列或可迭代对象，返回逆序或排序后的结果
  • reversed和zip一样返回一个可迭代对象，这个对象可用于for循环和例如join等方法，但是不能直接查看其内容，不能对其进行索引和切片操作，也不能直接对其调用列表的方法，需先用list进行类型转换
  • 与zip相同，reversed的返回结果经list之后变为空

break / continue

  • 循环中的if/break语句将循环分为两个部分
  • 直观的理解：第一部分负责设置，第二部分在循环条件为真时使用第一部分初始化的数据

循环中的else

  • 在for循环和while循环中都可使用else语句
  • 在循环正常结束时才调用，因此可以判断循环是否正常结束

    5.6 简单推导

  • 列表推导：从其他列表创建列表
  • 使用圆括号代替方括号不能实现元组推导，而是创建生成器
  • 字典推导：不同点在于使用冒号表示键值对

    5.7 pass/del/exec

pass

  • 用作占位符

del

  • 用于删除变量或数据结构的成员，但不能用于删除值
  • 在之前对于clear()的探讨中，对x = y,将x置为None或空列表/字典时，y并未发生变化，这是由于将x置为空只是修改了这一名称的关联，而并没有对值进行修改
  • 但当把y也置为空时，这一字典就漂浮在内存中，python编辑器会将其直接删除，这一机制称为垃圾收集

exec/eval
