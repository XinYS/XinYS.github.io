1.10 字符串

字符串表示str和repr
值到字符串的转换有两种不同的机制

repr和str分别对应以上两种机制

长字符串
使用三引号(‘’’或”””)来表示横跨多行的字符串
字符串可包含单引号双引号而无需转义

\可表示跨行
原始字符串
常规字符串在处理例如路径C:\nowhere时会出现问题

应注意引号需像通常那样进行转义，且原始字符串的最后一个字符不能是反斜杠

Unicode

  • Python字符串使用Unicode编码来表示文本，每个Unicode字符都用一个码点表示，即Unicode标准为每个字符指定的数字。
  • 在内存和磁盘中，所有对象都是由二进制数字编码表示的，八个为一组，即1字节。
  • 为与C语言进行相互操作等原因，python提供两种类型，不可变的bytes和可变的bytearray。
  • ASCII字符是用对应的数及其二进制编码定义的，Unicode码点是用整数定义的。
  • UTF-8使用了变长编码，进行单字节编码时，使用ASCII以与较旧系统兼容；对于不在此范围内字符，使用多个字符（最多6个）进行编码。
  • python默认使用UTF-8编码
  • 可直接创建bytes和str对象
