第七章 类

封装、隐藏

类的命名空间

  • 类的命名空间和其成员的命名空间的关系类似于前面提到的全局作用域和局部作用域的关系

继承

多重继承

  • 可以指定多个超类，位于前面的类的方法可以覆盖位于后面的类的方法
  • 多个超类的超类相同时，查找属性或方法的顺序为方法解析顺序（MRO）

接口和内省

派生、抽象基类

  • 抽象类是不能实例化的类，其职责是定义子类应该实现的一组抽象方法

重写构造函数

  • 构造函数：对象创建后自动调用

特性

  • 通过存取方法定义的属性称为特性

静态方法、类方法

  • 静态方法定义中没有参数self，可直接通过类来调用
  • 类方法的定义包含类似于self的参数cls，可通过对象来调用，但参数cls自动关联到类
