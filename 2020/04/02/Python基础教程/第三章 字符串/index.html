3.3 字符串格式设置 完整版

{字段名!转换标志:格式说明符}

  • 并非只能使用用于替换字段名的参数值本身，还可使用索引以及访问导入模块中的方法

若在最终结果中包含花括号，则使用两个花括号来表示
转换标志

  • 若设置了转换标志，将使用指定的函数将对象转换为字符串，再进行进一步的格式设置（默认为s）

格式说明符

  • 详细指定最终格式，包括格式类型、字段宽度、数的精度、显示符号、千分位分隔符、对齐和填充方式等

类型说明符

类型 含义
b 将整数表示为二进制数
c 将整数解读为Unicode码点
d 将整数视为十进制数进行处理，这是整数默认使用的说明符
e 使用科学表示法来表示小数（用e来表示指数）
E 与e相同，但使用E来表示指数
f 将小数表示为定点数
F 与f相同，但对于特殊值（nan和inf），使用大写表示
g 自动在定点表示法和科学表示法之间做出选择。这是默认用于小数的说明符，但在默认情况下至少有一位小数
G 与g相同，但使用大写来表示指数和特殊值
n 与g相同，但插入随区域而异的数字分隔符
o 将整数表示为八进制数
s 保持字符串的格式不变，这是默认用于字符串的说明符
x 将整数表示为十六进制数并使用小写字母
X 与x相同，但使用大写字母
% 将数表示为百分比值（乘以100，按说明符f设置格式，再在后面加上%）
宽度、精度、千位分隔符
- 宽度,(千位分隔符).精度
- 浮点数格式默认显示小数点后六位精度
符号、对齐、0填充
- 现对以上表达式进行说明
- 字符串和数的默认对齐方式不同，当指定一定宽度时，数字为右对齐，字符串为左对齐，为统一格式，
- 使用对齐说明符：<左对齐、>右对齐、^居中、=字符左对齐，数字右对齐；
- 在对齐方式前面可添加填充标志，可为零、加减号、空格、星号等；
（例如0^为居中0填充，=为符号左对齐、数字右对齐、中间使用\填充）
- 使用符号说明符为正数指定符号，-为默认，无符号、+为正号、空格为添加空格；
- #选项放在符号说明符和宽度之间，触发另一种转换方式，转换细节随类型而异，例如对二进制八进制等加上0b、0o前缀，对十进制数要求包含小数点，对于类型g保留小数后的零；
