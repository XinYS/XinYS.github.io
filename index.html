numpy_learn

#numpy笔记

  • numpy.random.random(size=None)
    • 产生[0.0, 1.0)之间的浮点数
  • numpy.random.uniform(low=0.0, high=1.0, size=None)
    • 生出size个符合均分布的浮点数，取值范围为[low, high)，默认取值范围为[0, 1.0)
  • numpy.random.rand(d0, d1, …, dn)
    • 生出size个符合均分布的浮点数，取值范围为[low, high)，默认取值范围为[0, 1.0)
  • numpy.random.randint(low, high=None, size=None, dtype=’I’)
    • 生成size个整数，取值区间为[low, high)，若没有输入参数high则取值区间为[0, low)
  • numpy.random.random_integers(low, high=None, size=None)
    • 生成size个整数，取值区间为[low, high], 若没有输入参数high则取值区间为[1, low]，注意这里左右都是闭区间
  • numpy.arange()
  • numpy.nonzero()
  • chararray.mean()
  • numpy.pad(a, (1, 1), mode=’constant’, constant_values=0)
  • chararray.max(), chararray.min()

数组取整

  • numpy.ceil(x,)
    向正无穷取整
  • numpy.floor(x,)
    向负无穷取整
  • numpy.trunc/fix(x,)
    截取整数部分
  • numpy.rint(x,)
    四舍五入到最近整数
  • numpy.around(x,)
    四舍五入到给定的小数位

  • chararray.size

    • 总元素个数

  • chararray.itemsize

    • 单个元素所占的内存

numpy.chararray.ndim

  • array的维数

numpy.allclose(a, b, rtol=1e-05, atol=1e-08, equal_nan=False)

  • 比较浮点数之间的近似相等，假设二者具有相同形状

numpy.array_equal(a1, a2)

  • 比较数组是否具有相同形状和数值

numpy.diag(v, k=0)

Parameters

  • v : array_like
    若v为2D数组，返回其第k对角线上的1D数组；
    若v为1D数组，返回以其为第k对角线的2D数组
  • k : int, optional
    默认值为0；

    Returns

  • out : ndarray
    The extracted diagonal or constructed diagonal array.

数组切片

数组运算

  • numpy.dot(a, b)
  • a @ b
    • 矩阵 向量乘法
  • a ** b
    数组之间的运算是元素一一对应进行运算的

numpy.unravel_index(indices, shape, order=’C’)

  • indices中的值对应于shape维度的数组展开为一维数组的下标，返回indices在shape维度下的下标
  • 默认为以行为主展开，indices可为整数或整数数组

    Parameters

    • indices : array_like
    • shape : tuple of ints
    • order : {‘C’, ‘F’}, optional
      • C(row-major) : 以行为主
      • F(column-major) : 以列为主

        Returns

    • unraveled_coords : tuple of ndarray
      与indices维度相同

numpy.newaxis

  • 在插入的位置增加一个维度

numpy.sort(a, axis=-1, kind=None, order=None)

  • 沿axis维度对输入数组进行排序
  • axis为None时，先将数组展开为1D，再进行排序

numpy.argsort(a, axis=-1, kind=None, order=None)

  • 与numpy.sort()类似的排序函数，返回的是排序的下标
  • 常与numpy.take_along_axis()等函数结合用于排序

numpy.take_along_axis(arr, indices, axis)

  • 迭代地沿着特定axis方向按照index匹配并返回数据数组的1D数据切片
  • 使用argsort、argpartition等沿指定axis方向返回索引的函数生成该函数的index参数
  • 常用于排序

numpy.expand_dims(a, axis)

  • 在axis的位置插入一个维度
  • axis可为整数或整数的元组，注意插入的维度不应该超出原数组的范围

numpy.argmax(a, axis=None, out=None)

  • 返回沿axis方向的最大值索引

numpy.tile(A, reps)

  • 返回将A重复reps次的数组
  • 如果reps维度为d，函数结果的维度为max(d, A.ndim)
  • 即reps和A中纬度低的一个会扩张维度到与另一个相同的维度

and, or, &, |

  • 在DataFrame的切片过程，要注意逻辑变量的使用,需要求得满足多个逻辑条件的数据时，要使用& 和|，在某些条件下用and/ or会报错
    ValueError : The truth value of an array with more than one element is ambiguous. Use a.any() or a.all()

numpy.copysign(a, b)

  • 将b中的符号赋予a

numpy.seterr(all=None, divide=None, over=None, under=None, invalid=None)

  • 设置对浮点错误的处理方式

    Parameters

all{‘ignore’, ‘warn’, ‘raise’, ‘call’, ‘print’, ‘log’}, optional
Set treatment for all types of floating-point errors at once:

  • ignore: Take no action when the exception occurs.
  • warn: Print a RuntimeWarning (via the Python warnings module).
  • raise: Raise a FloatingPointError.
  • call: Call a function specified using the seterrcall function.
  • print: Print a warning directly to stdout.
  • log: Record error in a Log object specified by seterrcall.
    The default is not to change the current behavior.

    Returns

  • old_settings: dict
    Dictionary containing the old settings.

numpy.intersect1d(ar1, ar2, assume_unique=False, return_indices=False)

  • 查找两个数组的交集
  • 返回经过排序的唯一值

numpy.datetime64()

  • 日期类型

numpy.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None, axis=0)

  • 返回start和stop之间均匀分布的向量
  • 默认为[start, stop]
  • 参数endpoint=False时为[start, stop)

numpy.fromiter(iterable, dtype, count=-1)

  • 从迭代器生成数组
  • count默认为-1，表示读取全部迭代器生成的数据

ufunc

  • 即universal function，这些函数逐元素作用于narray对象
  • 这些函数在对narray进行运算的速度比使用循环或者列表推导式要快很多

ufunc.reduce(a, axis=0, dtype=None, out=None, keepdims=False, initial=, where=True)

  • 沿axis(默认=0)方向应用ufunc,维度减1
  • 例如，numpy.add.reduce等同于numpy.sum

numpy.set_printoptions(precision=None, threshold=None, edgeitems=None, linewidth=None, suppress=None, nanstr=None, infstr=None, formatter=None, sign=None, floatmode=None, *, legacy=None)

  • 打印设置
  • 例，数组打印的默认最大元素个数为1000，可通过np.set_printoptions(threshold=np.inf)来取消限制

chararray.flags

  • 数组的内存信息
